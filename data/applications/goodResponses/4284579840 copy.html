Sehr geehrtes Canonical-Team,

ich bewerbe mich als Web Developer bei Canonical. In meiner täglichen Arbeit setze ich JavaScript, HTML und CSS ein und verbinde diese mit einem Full-Stack-Verständnis, um moderne, wartbare Webanwendungen zu liefern. Besonders wichtig sind mir klare Architektur, Performance und Accessibility.

Zu meinen Projekten zählen das Frontend einer Lotterie-Website für den Hauptsponsor eines Bundesligavereins, das ich mit HTML, CSS und JavaScript realisiert habe, sowie die Neugestaltung eines kompletten Buchungsprozesses für ein Campervan-Vermietungsunternehmen, den ich mit Angular so optimiert habe, dass Abbruchraten deutlich sanken und mehr Buchungen entstanden.

Ich nutze regelmäßig React und Angular im Frontend und setze serverseitig oft Node.js mit Express ein. TypeScript hilft mir, die Codequalität hochzuhalten. Für Tests setze ich auf Jest und Puppeteer, um robuste Regressionen und End-to-End-Absicherung zu erreichen.

Gern bringe ich meine Erfahrung ein, um die Web-Erlebnisse von Canonical weiter zu stärken, und freue mich auf ein Gespräch.

Mit freundlichen Grüßen
